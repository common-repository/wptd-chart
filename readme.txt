== Description ==

### The #1 Advance Chart Addon for Elementor

WPTD Chart is advanced elementor chart plugin. Convert easily data into chart. We gives you 4 models of charts example: pie chart, line chart, doughnut chart, bar chart. Also you can control chart legend details. In future we planned to give more chart types as free.

Pie chart, line chart, doughnut chart, bar chart and many features:

== Installation ==

### Installation

1. Upload 'wptd-chart' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now read [How of Use](http://plugins.wpthemedevelopers.com/wptd-chart/)

== Frequently Asked Questions ==

**Can I use the plugin without Elementor Page Builder?**

No. You cannot use without Elementor since it’s an addon for Elementor.

**Does it work with any theme?**

Absolutely! It will work with any theme where Elementor works.
